# Changelog

All notable changes to Umbrella will be documented in this file.

## [0.1.0] - 2024-01-15

### Added
- Initial project setup with .NET 9 and Next.js 14
- Docker and Kubernetes configuration
- PostgreSQL, Redis, and Neo4j database setup
- Service layer interfaces:
  - Integration Service
  - Data Ingestion Service
  - Metrics Service
  - Anomaly Detection Service
  - Root Cause Analysis Service
  - Prediction Service
  - Recommendation Service
  - Knowledge Graph Service
  - Health Score Service
  - Chat Service
- Frontend components:
  - Navigation
  - Dashboard
  - Anomalies page
  - Chat interface
  - KPI cards and charts
- API client and HTTP utilities
- Documentation and setup guides

### Planned for Future Releases
- Integration implementations (Salesforce, HubSpot, QuickBooks)
- ML models for anomaly detection and forecasting
- Neo4j graph implementation
- LLM integration for chat
- Production deployment configuration
